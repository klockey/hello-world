Old Days

In the old days (1989) of the web people were constrained by web languages. Css was not around yet so correctly locating the objects of html was done explicitly with html tables. Around 1995, Javascript which was made by Netscape helped make html more viable by making it easy to modify the html dynamically. Flash broke through around the same time making it easy to make moving images and clips of movies etc. Also, Css came around about this time as well giving developers a better way of structuring data.

Internet of Things

The internet of things developed in the 2000's. No longer were people confined to sitting at their desktops with a clunky computers. Now people could use mobile phones, tablets, laptops etc.

Conclusion

As time progresses web languages will have to be adapted to different machines. It's amazing to think html has survived all these years and has been modified for all its different uses. In order for the web to a viable construct in the world it must be flexible and dynamic.
